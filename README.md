# MySite

A simple static HTML site with CSS-only style.
Contains a brief overview of my portfolio and a little about me and what i like.

## Project structure

```
/MySite
├─ index.html
├─ css/
│  └─ body.css
│  └─ header.css
│  └─ index.css
├─ assets/
│  └─ imgs/
└─ README.md

```

## Development

-   Edit HTML in the project root
-   Place styles in css/styles.css
-   Keep images in assets/imgs/
